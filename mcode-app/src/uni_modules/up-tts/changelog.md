# 更新日志

## [1.0.0] - 2026-05-31

### 新增
- 首次发布UTS TTS语音合成插件
- 支持iOS平台（基于AVSpeechSynthesizer）
- 支持Android平台（基于TextToSpeech）
- 支持HarmonyOS平台（基于@ohos.ai.textToSpeech）
- 实现文本转语音基础功能
- 支持语速、音调、音量调节
- 支持多语言切换
- 支持朗读控制（播放、暂停、继续、停止）
- 支持朗读状态回调（开始、完成、错误、进度）
- 支持获取可用语音列表
- 支持自定义语音选择
- 提供完整的TypeScript类型定义
- 提供详细的使用文档和示例代码

### 平台特性
- iOS: 完整支持所有功能，包括暂停/继续和进度回调
- Android: 支持基本功能，进度回调需要API 23+
- HarmonyOS: 支持基本功能，支持在线和离线模式

### 文档
- 完整的README文档
- API接口文档
- 使用示例和最佳实践
- 平台差异说明
- 常见问题解答
